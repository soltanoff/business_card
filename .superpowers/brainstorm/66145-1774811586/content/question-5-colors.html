Цветовая схема

Сейчас — teal (#5cd7b8) акцент на тёмном. Для профессионально-строгого стиля есть варианты

A: Оставить текущий (teal + purple)

Тот же акцент, уже привычный. Gradient-заголовок teal→purple.

B: Steel blue — холодный и деловой

Синий акцент, slate-серые тона. Как корпоративный дашборд. Строго и надёжно.

C: Monochrome + один акцент

Почти чёрно-белый, teal только для ключевых элементов (имя, иконки). Максимальная строгость.

D: Teal→Blue gradient + neutral

Градиент для акцентов, нейтральный серый для текста. Современно, но сдержанно.
